个人博客搭建

之前就一直想做一个类似的个人网页，但是由于服务器和域名购买，html和css的搭建都有点麻烦，也不好零基础就搭建一个很美观的，这个事情就被一直搁置下来了。近期偶然发现了可以将网站部署到GitHub上，又有很多hexo的教程和模板，就尝试了下。

视频里讲的已经很详细了，我这里就写一下大致的步骤
1.安装node.js
2.安装hexo
3.进行本地配置
4.部署到云端
5.进一步优化

tips1：如果是win10系统的话建议安装Git Bash，可以搜索Git For Windows进行安装，用过linux之类系统的话不要太爽。
tips2：用git bash的话要修改用户名称和邮箱。

git config --global user.name "xxx"
git config --global user.email "xxx"

tips3：如果出现本地预览的时候hexo server启动成功但是加载不出来界面的话可以换一下端口，实测可行。
hexo s -p 3600

总的来说还是非常方便的，如果觉得GitHub速度过慢还可以部署到Gitee上。文档编写用的是markdown，有能力的话也可以做到非常美观。后期维护步骤就是

hexo clear
hexo g
hexo d
